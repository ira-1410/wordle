## Play the Game!
1. **Clone from directory**:
   ```
   git clone https://github.com/ira-1410/wordle
   cd wordle
   ```
   Open main.html directly in your browser
2. **Controls**
   - Only alphabet keys, ENTER and BACKSPACE is registered
   - ENTER to submit your guess.

## Key Features
1. **Keyboard and Onscreen Input**: Use your keyboard or click the on-screen keyboard to input.
2. **Hints**: Checking for correct, present and absent letters based on letters in the answer and number of duplicates.
3. **Unlimited words**: Play again unlimited times (until you get bored)

## Project Structure
- main HTML, CSS and JS file.
- words.js contains a list of 3K words to choose from. Sourced from [Github](https://gist.github.com/shmookey/b28e342e1b1756c4700f42f17102c2ff).
